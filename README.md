## DESCRIPTION
This script will install and configure NGNIX on a target CentOS 7.0+ Linux system.

EXAMPLE 1
InstallAndConfigureNgnix.sh

REQUIREMENTS:

1. CentOS 7.0+
2. Root or sudo permissions

REFERENCES:
https://www.tecmint.com/install-nginx-on-centos-7/
